คำแนะนำการตั้งค่า Environment Variables

สำหรับ Render.com

1. Edoclite Credentials

ตั้งค่าข้อมูลสำหรับเข้าสู่ระบบ Edoclite:

EDOCLITE_USER

ชื่อผู้ใช้สำหรับเข้าสู่ระบบ

EDOCLITE_PASS

รหัสผ่านสำหรับเข้าสู่ระบบ

2. Google Sheets Configuration

ตั้งค่าการเชื่อมต่อ Google Sheets:

GOOGLE_SHEET_ID

ID ของ Google Sheet ที่ต้องการเก็บข้อมูล

GOOGLE_SERVICE_ACCOUNT_JSON_B64

Service Account JSON ที่เข้ารหัส Base64

หรือใช้ GOOGLE_SERVICE_ACCOUNT_JSON สำหรับ JSON แบบ Raw

3. LINE Notify (ทางเลือก)

สำหรับการแจ้งเตือนผ่าน LINE:

LINE_NOTIFY_TOKEN

Token สำหรับส่งการแจ้งเตือนผ่าน LINE

ไม่บังคับ แต่แนะนำสำหรับการแจ้งเตือน

4. Flask Configuration

การตั้งค่าเพิ่มเติมสำหรับ Flask:

FLASK_SECRET_KEY

Secret key สำหรับ Flask session

PORT

Port สำหรับ Web server (Render ตั้งค่าอัตโนมัติ)
