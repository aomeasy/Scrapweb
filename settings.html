คำแนะนำการตั้งค่า Environment Variables ใน Render.com

1. Edoclite Credentials

EDOCLITE_USER = ชื่อผู้ใช้สำหรับเข้าสู่ระบบ Edoclite

EDOCLITE_PASS = รหัสผ่านสำหรับเข้าสู่ระบบ Edoclite

2. Google Sheets Configuration

GOOGLE_SHEET_ID = ID ของ Google Sheet ที่ต้องการเก็บข้อมูล

GOOGLE_SERVICE_ACCOUNT_JSON_B64 = Service Account JSON ที่เข้ารหัส Base64

หรือ GOOGLE_SERVICE_ACCOUNT_JSON = Service Account JSON แบบ Raw

3. LINE Notify (Optional)

LINE_NOTIFY_TOKEN = Token สำหรับส่งการแจ้งเตือนผ่าน LINE

ไม่บังคับ แต่แนะนำสำหรับการแจ้งเตือน

4. Flask Configuration

FLASK_SECRET_KEY = Secret key สำหรับ Flask session

PORT = Port สำหรับ Web server (Render จะตั้งค่าให้อัตโนมัติ)
